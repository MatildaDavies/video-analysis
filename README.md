# Analysing children's TV vs YouTube

This repository contains Python scripts that analyse videos for visual data, including motion, colour saturation and cut times. I used it to examine the visual differences between children's TV shows on YouTube (like Cocomelon and Blippi) compared with traditional broadcast television (like In the Night Garden and Bob the Builder). 

Read the final piece here: [https://www.thetimes.com/uk/education/article/youtube-videos-kids-television-xcxkv0cvx](https://www.thetimes.com/uk/education/article/youtube-videos-kids-television-xcxkv0cvx)

### Contents:
- Selection of video files used in the analysis, as examples
- Scripts to analyse visual elements of the videos
